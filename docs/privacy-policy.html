Política de Privacidad de SixAgentic

Última actualización: 16 de junio de 2026

SixAgentic es una aplicación cliente móvil que permite a los usuarios acceder a su propia implementación de HermesHQ. SixAgentic no es un servicio en la nube ni un proveedor de servicios; es exclusivamente una interfaz que se conecta al servidor que el usuario o su organización configuran y controlan.

1. SixAgentic no recopila ni almacena datos

SixAgentic no opera servidores propios, no recopila, no almacena y no tiene acceso a ningún dato del usuario. Toda la información — incluyendo credenciales, mensajes, archivos, y cualquier contenido intercambiado con los agentes de IA — se transmite directamente entre la aplicación y el servidor de HermesHQ que el usuario ha configurado. SixAgentic actúa únicamente como un cliente de conexión.

2. Autenticación y credenciales

Las credenciales de acceso (usuario y contraseña) se envían exclusivamente al servidor HermesHQ configurado por el usuario. SixAgentic almacena el token de autenticación de forma segura en el dispositivo utilizando los mecanismos nativos del sistema operativo (Keychain en iOS, EncryptedSharedPreferences en Android) para mantener la sesión activa. Este token nunca se envía a servidores distintos al que el usuario configuró.

3. Configuración del servidor

Al instalar SixAgentic, el usuario especifica la URL del servidor HermesHQ al cual conectarse. SixAgentic no incluye ningún servidor predefinido ni realiza conexiones a servicios de terceros para el procesamiento de datos.

4. Comunicaciones

Toda la comunicación entre la aplicación y el servidor HermesHQ del usuario se realiza a través de conexiones cifradas (HTTPS/TLS). SixAgentic no intercepta, almacena ni retransmite los datos a través de servicios intermedios.

5. Datos almacenados en el dispositivo

La aplicación almacena localmente en el dispositivo:

  • La URL del servidor configurada por el usuario
  • El token de autenticación (de forma segura)
  • El historial de conversaciones en caché para visualización offline

Estos datos se eliminan completamente cuando el usuario desinstala la aplicación. Ninguno de estos datos se envía a servidores externos.

6. Privacidad de los datos en HermesHQ

Dado que SixAgentic es solo un cliente, la privacidad y el manejo de los datos enviados a través de la aplicación están determinados por las políticas de privacidad de la implementación de HermesHQ que cada usuario ha desplegado. Le recomendamos consultar la política de privacidad de su organización o de su instancia de HermesHQ.

7. Datos analíticos o de telemetría

SixAgentic no recopila datos analíticos, de telemetría ni de seguimiento. No integramos SDKs de análisis de terceros ni herramientas de seguimiento de comportamiento.

8. Privacidad de menores

La aplicación no está dirigida a menores de 13 años. No recopilamos deliberadamente información de menores.

9. Cambios a esta política

Podemos actualizar esta política de privacidad periódicamente. Los cambios se publicarán en esta página.
